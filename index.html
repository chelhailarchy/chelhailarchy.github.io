Profile

I'm currently

I have been a freelance illustrator for six years and counting. I love to create traditional artworks using charcoal, ballpen, oil pastel, and watercolor. For digital, I usually use MediBang Paint, and I recently started using Adobe Creative suites like Photoshop, In Design, Illustrator, and Lightroom.

I'm also learning how to code; mainly, because I want to create my art website, and I preferably want to be a front-end software engineer so I can utilize my graphic designing skills for UI and UX design.

My interests and hobbies are watching anime, listening to music, coffee, anything about space and planets, horror films, and learning how to speak French.
